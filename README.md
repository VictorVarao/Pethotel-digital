# Pethotel-digital
O PetHotel Digital é um sistema de gerenciamento desenvolvido com o objetivo de modernizar e organizar a administração de hotéis para animais de estimação.
A plataforma permite que clientes realizem cadastro, registrem seus pets, façam reservas online, acompanhem o status da hospedagem, visualizem relatórios e efetuem pagamentos digitais.
O sistema também conta com um painel administrativo completo, onde a gestão do hotel pode controlar quartos, reservas, serviços adicionais, pagamentos, relatórios financeiros e atividades realizadas pelos funcionários.
O projeto foi estruturado com modelagem de banco de dados relacional, organização de tarefas no Trello, prototipação de interface no Figma e versionamento no GitHub, aplicando boas práticas de documentação e desenvolvimento de sistemas.
